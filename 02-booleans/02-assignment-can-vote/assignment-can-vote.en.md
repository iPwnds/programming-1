# Assignment

### `TASK`
Write a function `can_vote(age)` that checks whether a person with age `age` is allowed to vote.
Every person that is at least 18 years old can vote.

#### USAGE

```python
>>> can_vote(5)
False

>>> can_vote(17)
False

>>> can_vote(18)
True

>>> can_vote(35)
True
```
