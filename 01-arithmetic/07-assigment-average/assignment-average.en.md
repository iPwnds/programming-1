# Average

The average of two numbers $x$ and $y$ equals

$$
\frac{x + y}{2}
$$

### `TASK`

Write a function `average(x, y)` that computes the average of the two given values `x` and `y`.


#### USAGE

```python
>>> average(0, 0)
0

>>> average(0, 10)
5

>>> average(6, 8)
7

>>> average(1, 2)
1.5
```
